Why does Instagram collapse line breaks in captions?

Instagram's caption renderer strips consecutive blank lines down to a single line break, which is why a caption that looked well-spaced while you were typing it often turns into a dense wall of text once posted. The common workaround creators use is inserting an invisible character — like a Braille blank symbol or a dot — on otherwise empty lines, which tricks Instagram into preserving the spacing because the line technically isn't empty.

Separating hashtags from the main caption body serves a similar visual purpose: putting hashtags in their own clearly separated block, rather than buried mid sentence, keeps the caption itself readable while still getting the discovery benefit hashtags provide. Many creators place hashtags several lines below the caption using this same spacer trick, so they're present but visually de-emphasized.

Will the spacer characters be visible to readers?

The dot and dash options are visible but subtle. If you want completely invisible spacing, choose the "blank invisible lines" option, which uses a character that renders as empty space.

Does this work for Instagram Reels captions too?

Yes — the same line-break behavior applies to Reels, feed posts, and carousel captions.
